# Spring Data JPA Demo
添加了Hibernate的MetaModel生成器，并整合了一下lombok，主要想试一下JPA复杂查询。  

## Getting Started
因为MetaModel生成器原因，需要先用maven编译一次得到MetaModel类
```shell script
mvn compile
```
或是通过IDE生成
